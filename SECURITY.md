# Security Policy

## Supported Versions

The latest version of Memento AI is actively supported.

| Version | Supported |
|----------|-----------|
| Latest | ✅ |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately to the maintainers.

Please include:

- Vulnerability description
- Steps to reproduce
- Possible impact
- Suggested mitigation

Do not disclose vulnerabilities publicly until they have been resolved.

## Security Features

Memento AI is designed with privacy-first principles:

- Fully Offline
- CPU-Only Inference
- Local SQLite Storage
- No Cloud API Calls
- Local Document Processing
- Local AI Inference
